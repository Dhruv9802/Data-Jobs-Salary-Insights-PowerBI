This is a Power BI practice project built using randomly selected job-market data.
The dashboard focuses on computer-related jobs and visualizes:-

->Number of jobs available
->Hourly salary distribution
->Yearly salary trends
->Job availability across locations

The main purpose of this project is to demonstrate Power BI skills, data modeling, and visualization techniques.
